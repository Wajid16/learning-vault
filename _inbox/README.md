# _inbox — Temporary Drop Zone

This folder is the handoff point between you and the agent for each session.

## How to use

### 1. Transcript (`transcript.md`)
Paste the YouTube auto-generated transcript into `transcript.md`.
- No need to clean it up — raw captions with no punctuation are fine.
- The agent reads it, writes the notes, then **clears this file**.

### 2. Diagrams (`diagrams/`)
Drop any diagram/screenshot images the instructor drew or showed into the `diagrams/` folder.
- Name them however you like — the agent will rename them to the proper convention when embedding.
- The agent embeds them in the long note, then **deletes the source files** from this folder.

### 3. Video link
The agent **always needs the YouTube video link** — it isn't in the transcript.
Provide it in your chat message alongside the transcript, like:
> "New session: [course name] — [video title]. Link: https://youtu.be/..."

---

*This folder is intentionally kept mostly empty — it's a staging area, not storage.*
