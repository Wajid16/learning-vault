# learning-vault

Notes from courses/videos I've actually watched, written to be re-read — not a certificate wall. Each note is a distilled understanding of one video, cross-linked to related concepts in other courses, with a few self-test questions so revisiting it is active recall instead of re-reading a summary.

Why this exists: I'd finish a course on ML, or Pydantic, or LangChain, and a month later it felt like it never happened. This repo is the fix — one place, same format, connections made explicit instead of each course living in its own silo.

See [`_meta/workflow.md`](_meta/workflow.md) for how a note gets made.
