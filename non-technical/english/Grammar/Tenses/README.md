# Tenses

Source: [Learn all the Tenses in English: Complete Course](https://youtu.be/O9S70oJAivI) — Rebecca Ezekiel (~10hrs, all 12 English tenses, beginner → advanced).

These notes are a **full reference**, not the vault's usual one-screen distillation — the goal is a permanent "look this up forever" grammar reference, so each tense file goes deep: full form, every use-case, time markers, common confusions vs. other tenses, and examples. Still cross-linked and skimmable, just not length-capped.

## Note format (per tense)

- **TL;DR** — what the tense is for, in one line.
- **Form** — positive / negative / question, for all persons if irregular.
- **When We Use It** — every use-case the instructor gives, with her own examples.
- **Time Markers / Signal Words** — words that cue this tense.
- **Common Confusions** — vs. other tenses, cross-linked to those note files.
- **How This Connects** — links to other tense notes and `_meta/glossary.md` if relevant.
- **Self-Test** — questions that can't be answered from the title alone.
