## How these notes are structured

Each session note uses this template:
1. **TL;DR** — what the session was about and why it matters in AI Engineering.
2. **Section per topic group** — each term gets: What is it / Subtopics / Interview angle / Code examples.
3. **Added context** — supplementary information for AI Engineers (e.g., models, regex, configuration), clearly marked.
4. **Quick recap cheat-sheet** — one glance before the next session.
5. **Practice check** — exercises and questions to test yourself.
