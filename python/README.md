## How these notes are structured

Each session note uses this template:
1. **TL;DR** — what the session was about and why it matters
2. **Section per topic group** — each term gets: What is it / Subtopics / Interview angle / Typed examples / Practice programs
3. **Added context** — anything supplemented from outside the class, clearly marked
4. **Quick recap cheat-sheet** — one glance before the next session
5. **Practice check** — 2-3 questions to test yourself
